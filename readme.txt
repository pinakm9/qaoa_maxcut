jupyter notebook contains my demos for weighted max-cut using QAOA.
There are 5 demos for 5 different graphs:
1) simple chain
2) kite
3) 3-triangle
4) 3-triangle-2-antenna
5) 7-node Erdős–Rényi graph

modules contains my python modules for WeightedGraph and MaxCut classes which are used in the jupyter notebooks.


====================== Requirements =======================
modules:
	scipy version: 1.6.0
	numpy version: 1.20.1 <=== important!
	qiskit version: 0.16.4
	networkx version: 2.5
python:
	code in this repo has been tested with Python 3.8.3
